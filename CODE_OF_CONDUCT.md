# Code of Conduct

Dieses Projekt folgt dem [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Unser Standard

Wir erwarten von allen Beitragenden und Maintainern:

- respektvolle und konstruktive Kommunikation
- einen inklusiven, diskriminierungsfreien Umgang
- sachliche Diskussionen, auch bei Meinungsverschiedenheiten

## Nicht akzeptables Verhalten

Dazu gehören insbesondere:

- persönliche Angriffe, Beleidigungen oder Belästigung
- diskriminierende oder herabwürdigende Sprache
- wiederholtes Stören von Diskussionen oder Review-Prozessen

## Durchsetzung

Maintainer dürfen Kommentare, Issues, PRs oder Beiträge moderieren, entfernen oder schließen,
wenn sie gegen diese Regeln verstoßen.

## Verstöße melden

Bitte melde Vorfälle über:

- ein GitHub Issue (wenn öffentlich passend), oder
- direkt vertraulich an die Maintainer.

Alle Meldungen werden zeitnah und fair geprüft.
